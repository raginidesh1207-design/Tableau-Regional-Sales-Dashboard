# Tableau-Regional-Sales-Dashboard

## Overview

This Tableau dashboard compares sales performance across different regions using KPIs, maps, trend analysis, and sales summary tables. The dashboard enables users to identify top-performing regions, analyze sales trends, and support data-driven decision-making.

## Tools Used

* Tableau Public
* Sample Superstore Dataset
* Interactive Filters
* Geographic Maps
* KPI Analysis

## Key Features

* Regional sales comparison
* KPI monitoring (Sales, Orders, Customers, Products)
* State-wise sales visualization using maps
* Sales trend analysis using line charts
* Interactive filtering for region selection

## Business Insights

* The West region generated higher total sales than the East region.
* The East region achieved higher average sales per order.
* Geographic analysis highlights top-performing states and regional opportunities.
